URL Encoding

url=scheme://prefix.domain:port/path/filename

  • scheme-internet service:http/ https / ftp /file
  • prefix-domain prefix "www"by defualt
  • domain-internet domain name
  • port-port number at the host "80" by default
  • path-path at server root if omit
  • filename-document or resource
