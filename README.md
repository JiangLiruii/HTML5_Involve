- XHTML 更加严格的HTML,为了更好的让浏览器解析
  - 必须包括head, title, html body <!DOCTYPE>也是必须的,W3C的标准
  - 标签嵌套必须严格正确<b><i><b><i>会发生错误
  - 所有标签必须闭合,如果空白标签也必须使用/> 闭合
  - 标签必须是小写
  - 标签属性名必须小写
  - 属性值必须引号引起来
  - 禁止属性简写,比如 checked="checked" 不能直接写checked
